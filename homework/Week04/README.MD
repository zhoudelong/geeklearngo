# 电梯调度模型实现

## 项目结构



```text
project
 |_ funcs/          提出共给通用方法
 |_ models/         对象模型及方法定义，包含单元测试
 |_ tests_cases/    给定的几个测试案例
 |_ main.go         项目主入口，当前用来跑所有的测试案例
 |_ go.mod          模块，go版本声明

```


## 程序运行入口
```shell
go run ./main.go
```
